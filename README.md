# Autonobin


After being so deeply inspired by HTX studio's incredible auto-aiming bin system (At https://www.youtube.com/watch?v=H0XYANRosVo) , we have decided to completely from scratch create our own auto-aiming rubbish system, Autonobin, after being so encouraged and supported by the Mark Evison Foundation.

We have moreover decided to take on this project with only the material funds we needed we deduced in intial preparations, in order to help us become even more resilient Computer Scientist, to be able to employ new strategies to progress knowing that just throwing more money at the problem to bury it is not an option. Because of these self-imposed constraints, we have had to start the robot from a 4WD subassembly by Freenova, making use of 4 motors to give us True 4 wheel drive and pairing with a Raspberry Pi 5, as hardware controlling unit

The only reason why we are even able to take on this challenge is thanks to the gracious contributions of the Mark Evison Foundation and their tireless commitment to helping young people make their dreams come True, providing funds to ignite projects, passions just as Autonobin, truly helping to make dreams into reality.

This is why we decided to started fundraising £215.00 for the Mark Evison Foundation, exactly the amount they gave us to help make this porject a possibility, in order to help them expand their reach and brings this joy; this hope to even mroe young people today. In this repository we have made us of the Yolov5 model, adapting their well-labelled clear code for this high-performance model into a stripped-down version with only what we need this for this project. The original repository is at https://github.com/ultralytics/yolov5 . Thank you so much Ultralytics for such a powerful yet approchable tool.

In this repository, we have also made use of the Freenova 4WD car controls, making use of the motor modules (And all required higher-level modules) as well as the setup.py module. We truly thank freenova for making their code so accessible and interfacing  with the hardware so straightforward, whihc was very helpful for quickly developing the solution. The original repository is at  https://github.com/Freenove/Freenove_4WD_Smart_Car_Kit_for_Raspberry_Pi
